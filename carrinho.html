Adicionando seu Projeto

Com o repositorio configurarado, agora sim podemos enviar seus arquivos para todo o mundo ver o que você fez. Para isso digite o comando:

  • git push origin master

    • Com o comando anterior, enviamos as alterações para o repositório. Talvez abra um programa pedindo seu usuário e senha de login do GitHub. Dê ao programa essas informações. Como diz as imagens abaixo.

    Vá para a página do seu projeto no GitHub e seu arquivo já estará em seu repositório. Agora é só comemorar.
